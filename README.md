# Calculadora de Nekomamadas

Web simple para convertir montos en MXN a la unidad monetaria ficticia `nekomamada`.

## Regla de conversion

- `1 nekomamada = precio de referencia 3DS en MXN` del anio elegido (datos `year,price_mxn` de 2011 a 2026).
- El monto a convertir debe estar en **MXN** para que la division sea coherente con esa referencia.

## Como usar

1. Abre `index.html` en tu navegador.
2. Escribe un monto en MXN.
3. Elige el anio con el control deslizante.
4. Presiona **Convertir**.
